Allows you to filter bookmarks and history menus – just open menu and type what you want to find.

##### Hotkeys:
<table>
<tr><td>Escape               </td><td>show all / hide tooltip / close menu (default menu behavior)                                                      </td></tr>
<tr><td>(any modifier)+Escape</td><td>close menu (default menu behavior)                                                                                </td></tr>
<tr><td>Alt                  </td><td>close all menus (default menu behavior), only if <em>extensions.bookmarksMenuFilter.altCloseMenu</em> = true      </td></tr>
<tr><td>Alt, Alt             </td><td>close all menus (default menu behavior), only if <em>extensions.bookmarksMenuFilter.doubleAltCloseMenu</em> = true</td></tr>
<tr><td>Backspace            </td><td>delete last char                                                                                                  </td></tr>
<tr><td>Ctrl+Backspace       </td><td>delete last word                                                                                                  </td></tr>
<tr><td>Ctrl+C               </td><td>copy filter                                                                                                       </td></tr>
<tr><td>Ctrl+X               </td><td>cut filter                                                                                                        </td></tr>
<tr><td>Ctrl+V               </td><td>paste (append)                                                                                                    </td></tr>
<tr><td>Ctrl+I               </td><td>toggle match case                                                                                                 </td></tr>
<tr><td>Ctrl+R               </td><td>toggle RegExp                                                                                                     </td></tr>
<tr><td>Ctrl+A               </td><td>toggle "as is"                                                                                                    </td></tr>
<tr><td>Ctrl+Z               </td><td>undo                                                                                                              </td></tr>
<tr><td>Ctrl+Shift+Z, Ctrl+Y </td><td>redo                                                                                                              </td></tr>
<tr><td>F1                   </td><td>show hint                                                                                                         </td></tr>
</table>
